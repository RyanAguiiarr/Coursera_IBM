# Código de Conduta

## Nosso Compromisso

Nós nos comprometemos a fazer da participação neste projeto uma experiência livre de assédio para todos.

## Nossos Padrões

Exemplos de comportamento que contribuem para criar um ambiente positivo incluem usar linguagem acolhedora e inclusiva.

## Responsabilidades de Aplicação

Os mantenedores do projeto são responsáveis por esclarecer os padrões de comportamento aceitável.

## Escopo

Este Código de Conduta se aplica tanto dentro dos espaços do projeto quanto em espaços públicos.

## Aplicação

Instâncias de comportamento abusivo, assediador ou inaceitável podem ser relatadas aos mantenedores do projeto.

## Diretrizes de Aplicação

Os mantenedores seguirão estas Diretrizes de Aplicação Comunitária.

## Atribuição

Este Código de Conduta é adaptado do Contributor Covenant.
